# Transmission Line Diversion Portal

A Blazor Server application developed using ASP.NET Core and SQL Server for managing Transmission Line Diversion Requests.

## Features

- User Login
- Create Request
- Edit Request
- Delete Request
- Approve / Reject Request
- Upload PDF
- View PDF
- Dashboard Cards
- Status Badges
- Validation
- SQL Server Database
- Entity Framework Core

## Technologies Used

- ASP.NET Core Blazor
- C#
- SQL Server
- Entity Framework Core
- Bootstrap

## Future Improvements

- Email Notification
- Export to Excel
- Role Based Authentication
- Search & Filters
